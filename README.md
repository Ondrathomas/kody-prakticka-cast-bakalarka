# Analýza energetických profilů a klasifikace charakteru zátěže

Tento repozitář obsahuje zdrojové kódy k praktické části bakalářské práce. Cílem projektu je transformace surových energetických dat, výpočet fyzikálních parametrů a následná kategorizace odběrných míst.

## Struktura projektu
- `01_preprocessing.py` - Načtení dat, pivotace a filtrace objektů (threshold).
- `02_physical_calculations.py` - Výpočet jalového výkonu a účiníku (cos phi).
- `03_classification.py` - Kategorizace objektů na odporové, induktivní a smíšené.
- `04_visualization.py` - Generování grafů a statistik pro text práce.

## Použité technologie
- Python 3.x
- Pandas, NumPy (zpracování dat)
- Matplotlib, Seaborn (vizualizace)
